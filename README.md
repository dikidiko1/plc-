# PLC 기말과제

## 프로젝트 소개

미쯔비시 GXWORKS2와 GT DESIGNER3 를 사용하여 만든 미니게임입니다. 

## 사용한 명령어 

* 타이머 T
* 카운터 C
* 셋     SET
* 리셋   RST
* 무브   MOV(P)
* 비교연산
  * "="  같다
  * "<>" 같지 않다
  * '>"  작다
  * "<=" 작거나 같다
  * "<"  크다
  * "<=" 크거나 같다
* 사칙연산
  * "+" 더하기
  * "-" 빼기
  * "*" 곱셈
  * "/" 나눗셈
* 특수 릴레이
  * SM411 0.2초마다 깜빡
  * SM413 2초마다 깜빡
